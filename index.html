FEATURES

機能

同志のために用意された機能一覧

🎵

国歌の再生

ボイスチャンネルに参加してソ連国歌をリピート再生する。1%の確率でロシア国歌が流れることも。

📊

リスニング記録

1周ごとに再生回数と総再生時間を記録。自分がどれだけ国歌を聴いたか把握できる。

🏆

ランキング

全ユーザー横断で総再生時間のランキングを表示。誰が最も忠実な同志かが一目でわかる。

📜

歌詞の送信

ソ連国歌の歌詞をDMに送付する。覚えて声に出して歌うのだ、同志よ。

COMMANDS

コマンド一覧

スラッシュコマンドで操作する

/soviet いるVCに参加してソ連国歌をリピート再生する PUBLIC
/soviet-stop 再生を停止してVCから退出する PUBLIC
/lyrics ソ連国歌の歌詞をDMに送付する（30秒クールダウン） PUBLIC
/listeningreport 自分の再生回数・総再生時間をDMに送る PUBLIC
/ranking 総再生時間の全体ランキング上位10名を表示する PUBLIC
